Prices at LGI Homes latest project in Charlotte region start below $300K WSOC TV

August 24, 2024
blog

CHARLOTTE — A home builder that is among the most active in the Charlotte area is adding to its local portfolio with a new project in the western part of the region.

LGI Homes is building Pinnacle Estates, a 92-home community near Shelby in Cleveland County. The homebuilder is marketing the project as one with affordable homes.

Its website states that home prices there start at $264,900. LGI Homes announced late last month that it had begun sales for the project.
